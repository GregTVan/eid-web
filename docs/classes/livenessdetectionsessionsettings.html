controlFaceSimilarityThreshold

controlFaceSimilarityThreshold: number = 4.5
  • Defined in faceDetection.ts:360

Background: Once the initial aligned face is detected the session will start capturing "control" faces at interval set in the controlFaceCaptureInterval property until maxControlFaceCount faces are collected or the session finishes. These control faces are then compared to the aligned face to ensure that the person performing the liveness detection is the same person as the one on the aligned face. This prevents attacks where a picture is presented to the camera and a live person finishes the liveness detection.

defaultvalue

4.5
